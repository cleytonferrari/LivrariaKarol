Livraria Karol
======================

Exemplo de uso de ADO.Net com ASP.Net MVC 4, criando projetos para o repositorio, dominio, aplicação e Interface Gráfica;

## Exemplo usado ##
Criação de um catalago de livros para uma biblioteca.

## Dominio ##
Projeto onde fica as classes de negócio, os models.

## Repositório ##
Projeto onde fica a conexão propriamente dita com o ADO.Net, neste projeto fica os metodos responsaveis para lidar com o banco de dados.

## Aplicação ##
Neste Projeto fica as classes responsáveis pelos metodos de manipulação (CRUD) de um model.

## UI.Web ##
Projeto em ASP.Net MVC 4 que consome os metodos da aplicação.

## UI.MsDos ##
Projeto em Console Application que visa demonstrar o uso simples do consumo da camada de Aplicação.
